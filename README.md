# tullingedk/website

Source code to [tgdk.se](https://tgdk.se).

## Working with the project locally

Node.js and npm is required.

Install modules.

```bash
npm install
```
Start the development server.

```bash
npm start
```

Build production build.

```bash
npm run build
```
